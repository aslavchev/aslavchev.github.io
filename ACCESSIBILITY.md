# Accessibility Statement

This portfolio website is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

## Conformance Status

This website strives to conform to **WCAG 2.1 Level AA** standards.

## Accessibility Features

### Keyboard Navigation
- ✅ All interactive elements are keyboard accessible
- ✅ Skip link to main content (press Tab on page load)
- ✅ Visible focus indicators on all interactive elements
- ✅ Logical tab order throughout the site
- ✅ No keyboard traps

### Screen Reader Support
- ✅ Semantic HTML structure with proper headings (h1-h6)
- ✅ ARIA labels on all interactive elements
- ✅ ARIA landmarks for page regions
- ✅ Alt text on all meaningful images
- ✅ Decorative images marked with aria-hidden
- ✅ ARIA live regions for dynamic content

### Visual Design
- ✅ Sufficient color contrast (4.5:1 for normal text, 3:1 for large text)
- ✅ Text can be resized up to 200% without loss of functionality
- ✅ No reliance on color alone to convey information
- ✅ Focus indicators meet 3:1 contrast ratio
- ✅ Responsive design for all viewport sizes

### Motion & Animation
- ✅ Respects `prefers-reduced-motion` preference
- ✅ No auto-playing videos or audio
- ✅ Animations can be paused
- ✅ No flashing or strobing content

### Touch Targets
- ✅ All interactive elements meet 44x44px minimum size
- ✅ Adequate spacing between touch targets
- ✅ Works with voice control software

## Keyboard Shortcuts

| Action | Shortcut |
|--------|----------|
| Skip to main content | Tab (when focused on skip link) |
| Navigate through links/buttons | Tab / Shift+Tab |
| Activate link/button | Enter / Space |
| Navigate sections | Arrow keys (in navigation) |
