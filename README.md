# Portfolio Website

Modern portfolio website built with Next.js 16, React 19, and TypeScript. Lighthouse: 95+ performance, 100 accessibility, 100 SEO. WCAG 2.1 AA compliant.

**[View Live Site →](https://aslavchev.com)**

---

## Tech Stack

- Next.js 16 (App Router) + React 19
- TypeScript + Tailwind CSS v4
- shadcn/ui components
- GitHub Pages deployment

## Development

```bash
npm install        # Install dependencies
npm run dev        # Start development server
npm run build      # Production build
```

## Performance

- WebP optimization (1.2MB savings, 35.8% reduction)
- OG image: 797KB → 87KB (89% reduction)
- First Contentful Paint <1.5s
- Lazy loading, code splitting, resource hints

## Accessibility

- WCAG 2.1 AA compliant
- Full keyboard navigation
- ARIA labels, screen reader optimized
- 44px minimum touch targets

## Features

- Dark/light theme
- Mobile-first responsive design
- Schema.org structured data
- Automated sitemap updates
