  1. Open this HTML file in your browser (Chrome recommended)
  2. Press F12 to open DevTools
  3. Press Ctrl+Shift+P (or Cmd+Shift+P on Mac)
  4. Type "screenshot" and select "Capture node screenshot"
  5. Click on the OG image container (the dark card above)
  6. Save the screenshot as og-image.png in the public folder
  7. The image will be exactly 1200x630px (perfect for social sharing)

Pro tip: To use your actual headshot instead of the "AS" placeholder, uncomment the img tag in the HTML and comment out the placeholder div.
